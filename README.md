# Danintendo

Una consola de videojuegos hecha con Arduino para la asignatura de Tecnología Industrial de 1º de Bachillerato (2019) con tres juegos:
- Buscaminas
- Serpiente
- Asteroides

### Controles
#### Buscaminas
- Botones de dirección / joystick: Mover cursor
- Botón izquierdo: Colocar bandera
- Botón derecho: Descubrir casilla
#### Serpiente
- Botones de dirección / joystick: Cambiar dirección
#### Asteroides
- Botones de dirección / joystick: Mover nave
- Botón izquierdo: Disparar

### Componentes
- Arduino UNO
- Pantalla LCD 20x4 I2C
- Joystick
- 6x Resistencia 220 Ohmios
- 6x Pulsadores
